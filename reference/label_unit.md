# Label the time unit

An auxiliary function to return the label of the exposure time unit
used.

## Usage

``` r
label_unit(unit)
```

## Arguments

- unit:

  Character that indicates the unit of exposure of the sports-related
  health problem data.

## Value

A character vector that expresses the unit used for the rates, i.e. for
the athlete's time at risk, named `unit_timerisk`.
